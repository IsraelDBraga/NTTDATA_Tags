Surgimento da Internet

Voltar

A Internet surgiu nos anos 60, na época da Guerra Fria, nos Estados Unidos. O Departamento de Defesa americano pretendia criar uma rede de comunicação de computadores em pontos estratégicos. Assim surgia a ARPANET, uma rede ligada por um backbone que passava por debaixo da terra. A ARPANET era de uso restrito dos militares e pesquisadores na época. No Brasil, a conexão de computadores por uma rede somente era possível para fins estatais. Em 1991, a comunidade acadêmica brasileira conseguiu, através do Ministério da Ciência e Tecnologia, acesso a redes de pesquisas internacionais Foi apenas em 1995 que houve a liberação para uso comercial, ficando a cargo da iniciativa privada os serviços de provedor de acesso.

Como funciona o HTML

Voltar

Criado em 1991 por Tim Berners-Lee, no CERN, o HTML é uma linguagem de marcação são constituídas de códigos que delimitam conteúdos específicos, segundo uma sintaxe própria. O HTML tem códigos para criar paginas na web. Estes códigos que definem o tipo de letra, qual o tamanho, cor, espaçamento, e vários outros aspectos do site. No início era muito complicado aprender HTML, pois eram muitos comandos para fazer algo simples. A cada nova versão, o HTML fica mais fácil de utilizar, e adquire mais funções. Para criar e editar códigos em HTML é necessário qualquer editor de texto comum, como bloco de notas. Para testar os códigos, basta salvar o arquivo em formato .HTML e executar. Para o teste é necessário ter um navegador configurado como padrão. Não é necessária internet, pois o arquivo com os códigos esta na maquina onde esta sendo executado.

O que são Tags

Voltar

Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros. As tags são formadas por uma estrutura própria, iniciam com o sinal “menor que”, em seguida vem o nome daquele elemento e por fim, o sinal “maior que”. Podem ser dispostas em tags que precisam de fechamento e tags que fecham sozinhas (self-closing). O fechamento de uma tag será definido com uma barra (/), sendo que no caso das tags de auto fechamento, não há necessidade da presença desse caractere.

Tags Abordadas em aula

Voltar

Estrutura Básica

<html>Esta tag é o elemento básico da estrutura do HTML. Assim sendo, ela conterá todos os elementos do seu documento.

<head>Essa tag delimita o cabeçalho do documento.

<title>Essa tag define o título da sua página, o nome que aparecerá na sua aba, janela ou guia.

<body>É nessa tag que todos os elementos visíveis do seu site devem ser inseridos.

Algumas tags

<i> Faz com que o texto fique em itálico

<strong> Faz com que o texto fique em negrito

<img> Utilizada para colocar imagem no site

<input>Utilizado para criar controles interativos

<a>Utilizado para colocar links

<hr> Faz uma linha de quebra na página html

<sup> Usado para escrever um texto com letras menores e com o alinhamento acima das que do resto do conteúdo

<sup> Usado para escrever um texto com letras menores e com o alinhamento abaixo das que do resto do conteúdo

Formatação de texto

<h1> Usado para título

<h2> Usado para subtítulo

<h3> até <h6> Diferentes tamanhos e prioridades de texto

<blockquote> Citação

<small> Define um texto menor

<mark> Destaca o texto

<font> Especifica a fonte

<del> Representa uma parte do texto que foi excluída de um documento

<abbr> É o marcador utilizado para se inserir uma sigla em uma página HTML

<p>Parágrafo

Atributos

Voltar

class=”…” – Atribui uma classe ao elemento (uma classe pode ser utilizada para um ou mais elementos);

id=”…” – Atribui um id ao elemento (um id deve ser único, ou seja atribuído a um único elemento);

style=”…” – Permite incluir elementos CSS (estilos) dentro da tag;

type=”…” – Indica o tipo de controle a ser exibido. O tipo padrão é text, se este atributo não for especificado. Influencia diretamente na tag Input

src=”…” – Incorpora o recurso no documento atual no local da definição do elemento.

href=”…” – especifica a localização de um recurso da Web.
